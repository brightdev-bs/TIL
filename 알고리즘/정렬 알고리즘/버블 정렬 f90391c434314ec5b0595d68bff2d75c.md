# 버블 정렬

다음의 숫자들을 오름차순으로 정렬하는 프로그램

1 10 5 8 7 6 4 3 2 9

📌핵심 : 옆에 있는 값과 비교해서 더 작은 값을 앞으로 보내기!

```c
#include <stdio.h>

int main(void) {
	int i, j, tmp;
	int arr[10] = {1,5,10,8,7,6,4,3,2,9};
	for(i = 0; i < 10; i++) {
		for(j=0; j< 9 - i; j++) {
			if(arr[j] > arr[j+1]) {
				tmp  = arr[j];
				arr[j] = arr[j+1];
				arr[j+1] = tmp;
			}
		}
	}
	for(i = 0; i < 10; i++) {
		printf("%d ", arr[i]);
	}
	return 0;
}
```
