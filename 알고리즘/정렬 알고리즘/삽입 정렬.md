# 삽입 정렬

1 10 5 8 7 6 4 3 2 9

포인트 : 각 숫자를 적절한 위치에 삽입하자.

-필요할 때만 위치를 바꾸기 때문에 버블 정렬 및 선택 정렬보다 효율적

-거의정렬된 상태에서는 어떤 알고리즘보다 빠름.

→앞에 있는 것은 정렬이 되어있다고 가정하기 때문임.

```c
#include <stdio.h>

int main(void) {
	int i, j, tmp;
	int arr[10] = {1,10,5,8,7,6,4,3,2,9};
	for(i = 0; i< 9; i++) {
		j = i;
		while(j >= 0 && arr[j] > arr[j+1]) {
			tmp = arr[j];
			arr[j] = arr[j+1];
			arr[j+1] = tmp;
			j--;
		}
	}
	for(i = 0; i < 10; i++) {
		printf("%d ", arr[i]);
	}
}
```

시간 복잡도 : O(n^2)
