#### 배열의 출력  ####
배열을 출력할 때 for문을 주로 이용하는데 더 간단한 방법이 있다.<br>

    Array.toString(배열이름)
위의 메소드는 배열의 모든 요소를 ' [ 첫번째 요소, 두번째 요소, ... ] '와 같은 형식의 문자열로 만들어서 반환한다. <br>

배열의 이름을 자체로 넣으면 '타입@주소'의 형식으로 출력된다.

하지만 char배열은 println 메소드로 출력하면 각 요소가 구분자없이 그대로 출력된다.

    char[] chArr = { 'a', 'b', 'c', 'd' };
	System.out.println(chArr); // abcd가 출력된다.

#### System.arraycopy()를 이용한 배열의 복사  ####

arraycopy()를 사용하면 보다 간단하고 빠르게 배열을 복사할 수 있다. 그 이유는, 지정된 범위의 값들을 한 번에 통째로 복사하기 때문이다.

    for(int i = 0; i < num.length; i++) { newNum[i] = num[i]; }
    -> System.arraycopy(num, 0, newNum, 0, num.length);
	-> num[0]에서 newNum[0]으로 num.length개의 데이터를 복사
