#### 문자열의 비교 ####
문자열의 비교 방법에는 2가지가 있다.<br>
1 ) '==' 비교 연산자 : 두 문자열이 완전히 같은 것인지 비교한다.<br><br>
2 ) equals() 메서드 : 문자열의 내용이 같은지 비교한다.

    String str1 = "abc"
    String str2 = new String("abc");
	
	// 아래의 결과 값을 예측해보자.
    str1 = "abc"
    str2 = "abc"

<br>
**[정답]** <br>
str1 = "abc" : true <br>
str2 = "abc" : false
