# 버전 관리

1 ) 자신의 이름과 이메일 등록하기

```c
git config --global user.name "Mango"
git config --global user.email "brightdev.bs@gmail.com"
```

2 ) 저장소 만들기 / 깃 상태 확인하기

```c
git init
git status
```

3 ) 스테이지에 올리기

```c
git add "example.txt"
```

4 ) 커밋하기

  -m을 이용해서 커밋 메세지를 정할 수 있다.

```c
git commit -m "learning github"
```

5 ) 스테이징과 커밋 동시에 하기

```c
git commit -am "add and commit"
```

6 ) 커밋 정보 확인하기

```c
git log
```

7 ) 최근 버전과 작업 폴더의 수정 파일간의 차이를 확인할 때

```c
git diff
```

8 ) 작업 트리에서 수정한 파일 되돌리기

```c
git checkout
```

9 ) 가장 최근 커밋 취소하기

```c
git reset HEAD^
```

10 ) 지정한 커밋으로 이동하고 이후 커밋 취소하기

```c
git reset --hard 커밋 해시
```

11 ) 지정한 커밋 해시의 변경 이력을 취소하기

```c
git revert 커밋 해시
```
