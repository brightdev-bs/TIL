# 지역 저장소와 원격 저장소 연결하기

지역 저장소를 원격 저장소와 연결하기 위해서는 다음의 명령어를 사용한다.

```c
git remote add origin "Repo 주소"

git remote -v // 연결되었는지 확인
```

원격 저장소와 일단 연결이 되면 원격 저장소에 코드 파일을 올리는 과정이 필요하다

이를 push한다고 한다.

```c
git push -u origin master
//일단 위와 같이 origin의 master브랜치와 연결을 하면 그다음부터는 다음과 같이 간결해진다.
git push
```

로컬에 있는 파일이 최신 상태가 아닌경우 원격 저장소의 최신 데이터를 가져오는 과정이 필요하다. 

이를 pull한다고 한다.

```
git pull origin master
```
