[PHP 기본 문법]

> PHP의 기본 틀은 다음과 같다.

```php
<?php

?>
```

> 주석은 //와  /* */을 이용한다.

> 변수 선언은 $를 이용 / 출력은 print 및 echo

```php
<?php
	$x = "Hello World";
	echo $x;
?>
```

> 변수의 데이터 형식을 알고 싶다면 gettype를 이용한다.

```php
<?php
	$x = 1;
	$y = "Hello World"
	echo gettpye($x);
?>
```

 변수의 타입 : 정수 / 실수 / 문자열 / 불린형 / 객체 / 배열

> if-else / while / for 문은 다른 언어와 비슷하게 작동한다.
