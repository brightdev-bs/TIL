PHP (Hypertext Preprocessor)란?

서버쪽에서 구동되는 언어로 사용자의 입력 데이터를 데이터베이스나 파일에 저장하고, 저장된 정보를 불러와서 웹, 앱으로 전송하는 등의 일을 한다.

---

<PHP는 왜 필요한 걸까?>

일반적으로 HTML을 이용해서 홈페이지를 만들 수 있다.

 —> HTML만 사용해서 만들어진 웹 사이트를 정적 웹사이트라고 한다.

HTML과 함께 JavaScript를 사용해서 클라이언트 스크립트를 작성할 수 있다.

클라이언트 스크립트란 클라이언트의 웹 브라우저에게 해석되는 스크립트를 말한다.

여기서 중요한 것이 웹 서버는 클라이언트가 요청하는 소스 코드를 클라이언트에게 전송하는 역할만 한다. 

서버에서 특정 작업을 거쳐 HTML 코드로 변환해서 클라이언트로 전달하는 언어를 서버 스크립트라고 한다. 서버 스크립트는 고정된 내용이 아니라 클라이언트가 요청할 때 마다 동적으로 작동하기 때문에 실시간으로 변경되는 웹 사이트를 처리할 수 있다. 

 —> 실시간으로 변경되는 웹 사이트를 동적 웹 사이트라고 한다.

 —> 서버 스크립트에는 PHP, JSP 등이 있다.
